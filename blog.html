1. What is the Document Object Model (DOM)?

When a web page is loaded, the browser creates a Document Object Model of the page.The HTML DOM model is constructed as a tree of Objects

2. How do you select an element from the DOM?

In JavaScript, we can select an element from the DOM (Document Object Model) using various methods provided by the document object.Example given below:

  • Selecting by ID: document.getElementById('elementId');
  • Selecting by Class Name: document.getElementsByClassName('className');
  • Selecting by Tag Name: document.getElementsByTagName('div');
  • Selecting by CSS Selector: document.querySelector('.className');
  • Selecting Multiple Elements by CSS Selector: document.querySelectorAll('.className');
3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation in the context of the DOM is a technique where we attach a single event listener to a parent element to manage events triggered by its child elements. Instead of adding event listeners to each individual child element, the parent element listens for events that bubble up from the children.

Event delegation is useful because it enhances performance by reducing the number of event listeners needed in the DOMâ€”allowing a single listener on a parent element to manage events for all its child elements. This is particularly beneficial in applications with many elements, as it minimizes memory usage and improves efficiency.

4. How do you manipulate an element's attributes and styles using the DOM?

We can manipulate an element's attributes and styles using various methods in the DOM.

  • Attribute Manipulation:
  • Get an Attribute: element.getAttribute('attributeName');
  • Set an Attribute: element.setAttribute('attributeName', 'newValue');
  • remove an Attribute: element.removeAttribute('attributeName');
  • Style Manipulation:
  • Inline Styles: element.style.backgroundColor = 'blue';
  • Adding Classes: element.classList.add('newClass');
  • Removing Classes: element.classList.remove('oldClass');
